Project Information

  • Role: UI/UX Designer & Frontend Developer
  • Client: State Board for Community & Technical Colleges (SBCTC), Washington, USA

Project Overview

Developed a user-centric online admissions portal for the Washington State Board for Community and Technical Colleges (SBCTC), enhancing the application process for over 337,000 students annually.

Category

Challenges :

  • Designing a centralized admission portal to serve 34 colleges with different branding requirements.
  • Ensuring WCAG 2.1 AA compliance for a diverse applicant pool, including users with disabilities.
  • Reducing high application drop-off rates due to complex workflows and long forms.
  • Handling large-scale traffic (350K+ applicants annually) while maintaining performance.
  • Balancing stakeholder requirements across multiple campuses with varied workflows.

Contributions :

  • Designed and developed a mobile-first, accessible frontend using semantic HTML, ARIA roles, and keyboard interactions.
  • Created wireframes and interactive prototypes in Adobe XD, delivering annotated specs with accessibility guidelines.
  • Improved application experience with autosave-enabled forms, streamlined navigation, and program comparison tools.
  • Integrated OKTA SSO, document preview, and deadline filtering for a smoother user journey.
  • Conducted usability testing across campuses, incorporating feedback into iterative design improvements.
  • Built a scalable design system to support 30+ college-branded sites with consistent UI.
  • Collaborated on performance optimizations (code-splitting, lazy loading) to cut initial load time by 40%.
  • Performed comprehensive accessibility testing with NVDA, VoiceOver, axe DevTools, and Lighthouse.
  • Reduced rework by providing developer-ready assets with detailed interaction and accessibility documentation.

Impact:

  • Boosted application completion rate by 25%, reducing form abandonment.
  • Lowered support tickets by 40% thanks to accessible, intuitive UI.
  • Accelerated portal load times by 40%, improving performance across devices.
  • Enabled consistent, scalable UI across 34 colleges, reducing design debt.
  • Supported 350K+ applicants annually, improving access to higher education.
